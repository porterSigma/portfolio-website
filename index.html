Memory Manager

A custom C++ memory management system designed to efficiently handle frequent allocations of small objects of varying sizesâ€”a scenario typical in game engines.

The system features a global MemoryManager that delegates requests to a SmallObjectAllocator, which uses fast bitwise operations to select the correct FixedAllocator. Memory is organized into chunks and blocks, enabling rapid reuse and minimizing fragmentation.

  • Global MemoryManager with SmallObjectAllocator delegation
  • Fast bitwise operation for FixedAllocator selection
  • Chunk and block organization for memory efficiency
  • Convenient macros (MM_NEW, MM_DELETE)
  • Automatic fallback to system allocator for large objects

Ghost'n Castle

Ghost'n Castle is a 2D mobile game inspired by the classic arcade title Pang. The player controls a ghost moving horizontally across the screen, dodging obstacles and casting spells to defeat enemies.

Each enemy, once hit, splits into smaller ones until fully eliminated. The project was developed in Unity with a focus on responsive mobile touch controls, scalable difficulty management, and efficient object pooling for optimized performance on Android devices.

One of the project's goals was also to learn how to use AI as a method of graphic and sound innovation, experimenting with procedural generation techniques.
